Use the Darwin Website Design System Skill in this folder.

Task: Build the first static version of Darwin Hernandez’s personal website using the mockups and instructions in SKILL.md.

Start with:
1. Read SKILL.md.
2. Inspect the mockups in assets/mockups.
3. Create design tokens for colors, typography, spacing, and cards.
4. Build reusable components.
5. Implement Home, Writing, Projects, About, and Now.
6. Use data files for essays, projects, playlists, and now items.
7. Link writing cards to Substack and project cards to GitHub.
8. Report any assumptions before finalizing.

Do not invent a new visual direction. Preserve the calm editorial Wayfinder identity.
